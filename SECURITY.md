# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| Latest Release | ✅ Yes |
| Older Versions | ❌ No |

## Reporting a Vulnerability

Security issues should be reported responsibly.

### How to report
- Do NOT open a public GitHub issue.
- Contact the maintainer with proper details and screenshots if possible.

### Response Time
- Initial response within 48–72 hours.
- Confirmed issues will be fixed quickly.

Thank you for helping improve the security of this project.
